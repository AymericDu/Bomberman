## Introduction

This application simulates the Bomberman Game

## Rules  

You can launch the game with 2 players

Player 1  : Z-S-Q-D and SPACE to drop a bomb   
Player 2  : Fleches and ENTRER to drop a bomb

## Execution  

$ mvn install  
$ mvn exec:java
